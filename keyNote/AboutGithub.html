Github使用筆記

建立專案

  1. 點擊New repository
  2. 輸入專案(存儲庫)名稱
  3. 相關設定
  4. 建立專案(存儲庫)

git clone

  1. 取得你的網址 EX： https://github.com/LiaoZike/liaozike.github.io.git

  2. 開啟CMD (命令提示字元)

  3. 切到你要抓的目的地資料夾

  4. 輸入指令

    git clone https://github.com/LiaoZike/liaozike.github.io.git

  5. 你的資料夾裡面就會多出一個資料夾，檔名為你儲存庫的名稱。

推到遠端

  1. CMD路徑切到那個資料夾裡面 EX:/TODO

  2. 輸入指令

    git add .

  3. 輸入指令

    git commit -m "Uploading code"

  4. 輸入指令

    git push

  5. 過程可能需要登入驗證

git pull
