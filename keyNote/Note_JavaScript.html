四、jQuery與css互動

套用在選擇器上

  1. css() :使用目前css
  2. addClass() :套上css標籤
  3. hasClass() :檢查是否有套上css標籤
  4. removeClass() :刪除css標籤
  5. toggleClass() :有套用css移除；若無套用套上css(反向)

五、jQuery與DOM互動

  1. html() :可設定含有HTML標籤的內容
  2. text() :純文字
  3. prepend() :加入別人家庭排在最前面
  4. append() :加入別人家庭排在最後面
  5. before() :跳過家庭排在最前面
  6. after() :跳過家庭排在最後面
  7. remove() :刪除整個指定(div)全部內容
  8. empty() :清空指定(div)內容，元素還是存在
  9. replaceWith() :取代內容

六、jQuery事件

A.滑鼠事件

  1. click() :按下並放開滑鼠左鍵 觸發
  2. dbclick() :當連結按下並放開滑鼠左鍵 觸發
  3. mousedown() :當按下滑鼠左鍵但還沒放開時 觸發
  4. mouseup() :當按下滑鼠座建在放開時 觸發
  5. mouseover() :當滑鼠滑到元件時 觸發
  6. mouseout() :當滑鼠滑出元件時 觸發
  7. mousemove() :當滑鼠移動時

B.鍵盤事件

  1. keypress() :按下按鍵 觸發
  2. keydown() :比keypress快一點觸發 ，但有些網頁只會被觸發一次
  3. keyup() :放開按下的按鍵時 觸發

C.網頁/視窗事件

  1. ready() :網頁DOM載完 觸發
  2. load() :下載網頁後 觸發
  3. resize() :瀏覽器大小改變時 觸發
  4. scroll() :拉瀏覽器捲軸時 觸發
  5. unload() :按某個連結離開目前頁面時 觸發

D.表單事件

  1. submit() :表單送出時 觸發
  2. reset() :重設表單時 觸發
  3. change() :表單欄位改變內容或選取狀態時 觸發
  4. focus() :選取或移動某個表單欄位時 觸發
  5. blur() :取消選取或離開某個表單欄位 觸發

七、jQuery特效

基本特效

  1. show() :顯示元素
  2. hide() :隱藏元素
  3. toggle() :每次執行就會切換與隱藏元素

滑動特效

  1. slideDown() :元素向下滑動
  2. slideUp() :元素向上滑動
  3. slideToggle() :每次執行就會切換元素向上或向下滑動

淡入淡出特效

  1. fadeIn() :元素淡入顯示
  2. fadeOut() :元素淡出顯示
  3. fadeTO() :元素淡化到顯示的透明度
  4. fadeToggle() :每次執行就會切換元素淡入或淡出顯示

        <img src="https://scontent.frmq2-2.fna.fbcdn.net/v/t1.15752-9/278850137_521225272998885_7112229198378045226_n.jpg?stp=dst-jpg_p100x100&_nc_cat=101&ccb=1-5&_nc_sid=4de414&_nc_ohc=I5OgwCDOfK4AX9EsUnY&_nc_ht=scontent.frmq2-2.fna&oh=03_AVK15_mmOL5s3SRCXU9lLD3G2Pj8eNHcFNAwk3Tm32waMg&oe=628809F0" width=50px alt="">
        
    </div>
    <script src="https://code.jquery.com/jquery-3.6.0.js"></script>
    <script>
        let mouseX = 0
        let mouseY = 0
        let offsetX = 0
        let offsetY = 0
        let isDown = false
        var div = document.querySelector(".mydiv");
        
        div.addEventListener('mousedown', function(e) {
          isDown = true
          mouseX = e.pageX
          mouseY = e.pageY
          document.addEventListener('mousemove', move)
        });
        
        document.addEventListener('mouseup', function(e) {
          if (isDown) {
            offsetX += e.pageX - mouseX
            offsetY += e.pageY - mouseY
          }
          isDown = false
          document.removeEventListener('mousemove',move)
        });
        
        
        function move(e) {
          if (isDown) {
            const dx = e.pageX - mouseX
            const dy = e.pageY - mouseY
            div.style.transform = `translate(${offsetX + dx}px,${offsetY + dy}px)`;
          }
        }
        
    </script>
</body>
</html>
